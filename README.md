# Hyperb

Hyperb is the [hyper.sh](https://hyper.sh) API client ruby gem.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'hyperb'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hyperb


## Configuration

```ruby
	client = Hyperb::Client.new(access_token: 'AC')
```
